Latest news

Bench4BL (benchmark and reproducibility study for bug localization study) accepted @ISSTA

Our paper on bug localization benchmark, entitled "Bench4BL: Reproducibility Study on the Performance of IR-based Bug Localization" has been accepted for the 27th International Symposium on Software Testing and Analysis (ISSTA 2018), written by Jaekwon Lee, Dongsun Kim, Tegawendé F. Bissyandé, Woosung Jung and Yves Le Traon.

FaCoY (code-to-code search tool) accepted @ICSE2018

Our paper on code-to-code search, entitled "FaCoY – A Code-to-Code Search Engine" has been accepted for the 40th International Conference on Software Engineering (ICSE 2018), written by Kisub Kim, Dongsun Kim, Tegawendé F. Bissyandé, Eunjong Choi, Li Li, Jacques Klein, and Yves Le Traon.

New Member Joined

Kisub Kim has joined on 1st of August, 2017. He will work on code indexing and search.

CoCaBu (our code search tool) accepted @EMSE

Our recent paper entitled "Augmenting and Structuring User Queries to Support Efficient Free-Form Code Search" has been accepted for Empirical Software Engineering Journal, written by Raphael Sirres, Tegawendé F. Bissyandé, Dongsun Kim, David Lo, Jacques Klein, Yves Le Traon.

Patch study paper accepted @ISSTA 2017

Our recent paper entitled "Impact of Tool Support in Patch Construction" has been accepted for the 26th International Symposium on Software Testing and Analysis (ISSTA 2017), written by Anil Koyuncu, Tegawendé F. Bissyandé, Dongsun Kim, Jacques Klein, Martin Monperrus, and Yves Le Traon.
